# React Props and State Cart Demo

This project is a simple React application that demonstrates the use of **Props** and **State** to manage data between components.

The application displays a list of products and allows users to add items to a cart. It helps beginners understand how React components communicate using props and how state is used to manage dynamic data.

## 🚀 Features
- Display product list
- Add items to cart
- Manage cart items using React state
- Component communication using props
- Simple and clean UI with basic CSS

## 🛠️ Technologies Used
- React JS
- JavaScript (ES6)
- HTML
- CSS

  
## 📚 Learning Purpose

This project is built for learning how **React Props and State work together** to build interactive UI components.
